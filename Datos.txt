Proyecto Palabras Sentimientso :  Ortony’s  Affective  Lexicon  
				(Ortony, Clore, & Collins, 1988)

Bases de Palabras : WordNet, Google Knowledge Graph Search

Interesante Paradigma : Noetic NLP

--> N-grams mirar porque son muy usados, son modelos de markov linguistas


Estructura:

Recogida de datos : Twiter, solo 7 dias hacia atras 

Preprocesado de datos: 
	- Eliminar datos no en ingles ( tanto con un detector como eligiendo cuentas inglesas)
	- Tratar Retweets, o eliminar o nueva variable contador de rt
	- Pasar a codificacion ascii
	- Reacer abreviaciones con diccionarios de slang etc
	--> Simple truco para tratar negaciones, añadir un NOT_ a toda palabra con
		una negacion antes de ella, hasta el siguiente punto o coma
		" Esto no es bueno"--> "esto no NOT_es NOT_bueno"

Analisis:
	Para cada twit se necesita saber el objetivo de la frase ( A quien se refiere )
	y el sentimiento que proyecta sobre dicho objetivo

	- Sentimiento : Binario (bien mal) ? multinomial?
	
	- Posibles algoritmos : 
		- Naive bayes como clasificador general ( algo pobre)
		- Suppor vector machine
		- Uso de lexicon (base de datos con sinonimos de palabras, posiblemente muy util)

	- Indicadores tipicos para algoritmos : 
		- Tabla de contingencias
		- Sensibilidad y precision ...
		- F-measure : combinan sensibilidad y precision
		- Crosvalidation ...
